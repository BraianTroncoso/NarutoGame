# NarutoGame
Nuestro personaje "Naruto" debe defender la aldea de la hoja una vez mas y para lograrlo tiene que evitar la mayor cantidad de shuriken por 4 niveles. 
¿Que esperas para probarlo?
En cada nivel se aumentara la dificultad como tambien asi la velocidad de los "shurinken", en caso de que nuestro jugador gane el juego saldrá un mensaje emergente indicando que el usuario ganó y le mostrará su puntaje obtenido.
Juego realizado en Java usando el paradigma OOP
